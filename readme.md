# 🎨 Projeto: Desenvolvimento Web - Museu Nacional

Este repositório contém um projeto de site desenvolvido com **HTML5** e **CSS3** representando uma página institucional fictícia do **Museu Nacional**. A proposta é apresentar exposições, vídeos, localização, galeria de fotos e uma seção para agendamento de visitas.

🔗 **Repositório:**  
[https://github.com/felipedev2025/desenvolvimentowebmuseu](https://github.com/felipedev2025/desenvolvimentowebmuseu)

---

## 📁 Estrutura de Diretórios

```
/index.html
/css/
  ├── normalize.css
  └── estilo.css
/img/
  ├── museu.png
  ├── depoimento.png
  ├── imagem1.jpg
  ├── imagem2.jpg
  ├── imagem3.jpg
  └── imagem4.jpg
```

---

## ✨ Funcionalidades

- ✅ Navegação entre seções do site
- ✅ Vídeo do museu incorporado do YouTube
- ✅ Mapa de localização via Google Maps
- ✅ Lista de exposições em destaque
- ✅ Formulário para agendamento de visitas
- ✅ Galeria de fotos
- ✅ Página responsiva com layout organizado

---

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- Normalize.css
- Google Maps Embed API
- YouTube Embed

---

## 🚀 Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/felipedev2025/desenvolvimentowebmuseu.git

# Acesse a pasta
cd desenvolvimentowebmuseu

# Abra o arquivo index.html no navegador
```
